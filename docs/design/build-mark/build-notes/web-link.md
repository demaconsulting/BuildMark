### WebLink

#### Purpose

`WebLink` is a record in the BuildNotes subsystem that represents a hyperlink. It is used by
`BuildInformation` to carry the optional complete-changelog link that appears at the end of
the rendered markdown report.

#### Data Model

**LinkText**: `string` — human-readable link label shown in the report.

**TargetUrl**: `string` — fully-qualified URL that the link points to.

#### Key Methods

N/A — `WebLink` is an immutable data record with no methods beyond those auto-generated by C#.

#### Error Handling

N/A — immutable data record with no methods that detect or propagate errors.

#### Dependencies

N/A — `WebLink` has no dependencies on other software items; it is a simple record of two
strings.

#### Callers

- **BuildInformation** — holds a `WebLink?` as `CompleteChangelogLink`
- **RepoConnectors** — connectors construct the `WebLink` from baseline and current tags
