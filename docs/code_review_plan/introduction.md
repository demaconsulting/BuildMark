# Introduction

This document defines the formal code review plan for BuildMark.

## Purpose

To define review-sets and the evidence required to demonstrate that each software item
has been reviewed according to the project's compliance requirements.

## Scope

This document covers all review-sets defined in `.reviewmark.yaml` for BuildMark. The review
plan is automatically generated by the ReviewMark tool from the `.reviewmark.yaml` configuration
and included in the compiled document as `generated/plan.md`.
