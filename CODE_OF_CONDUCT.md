# Code of Conduct

## Community Guidelines

BuildMark is an open source project maintained by DEMA Consulting. We are committed to providing a welcoming and
professional environment for all contributors and users.

## Expected Behavior

All participants in the BuildMark community are expected to:

- Treat others with respect and professionalism
- Provide constructive feedback and accept it gracefully
- Focus discussions on technical merit and project improvement
- Help create a positive learning environment
- Acknowledge different skill levels and experiences

## Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment, discrimination, or hostile communication
- Personal attacks or inflammatory language
- Sharing private information without permission
- Disruptive or off-topic discussions
- Any conduct that would be inappropriate in a professional setting

## Reporting Issues

If you experience or witness unacceptable behavior, please report it by:

- Opening a confidential issue on GitHub
- Contacting the project maintainers directly

All reports will be handled with discretion and confidentiality.

## Enforcement

Project maintainers are responsible for enforcing these guidelines. Actions may include:

- Warning the individual about their behavior
- Temporary or permanent ban from project spaces
- Other actions as deemed appropriate

## Scope

These guidelines apply to all BuildMark project spaces, including:

- GitHub repositories (issues, pull requests, discussions)
- Project documentation
- Any official project communication channels

## Questions

For questions about these guidelines, please open a discussion on GitHub or contact the maintainers.
